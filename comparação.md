## Parte 5 – Comparação com Sistema Didático

Ao comparar o sistema real com o sistema didático analisado anteriormente, é possível perceber algumas diferenças importantes.

Em relação à arquitetura, o sistema real apresenta uma estrutura mais organizada e próxima de um ambiente profissional, enquanto o sistema didático é mais simples e voltado para aprendizado.

A coesão no sistema real é maior, pois as funcionalidades estão mais bem integradas. Já no sistema didático, a organização pode ser mais básica.

O acoplamento no sistema real aparenta ser mais controlado, enquanto no sistema didático pode ser mais alto devido à simplicidade da implementação.

Em termos de organização, o sistema real possui melhor estruturação de elementos, categorias e navegação. O sistema didático é mais limitado.

Por fim, o sistema real apresenta maior flexibilidade e possibilidade de evolução, enquanto o sistema didático é mais restrito.
